# UC3M-Statistical-Learning
This project consisted in two parts.

Part 1: Statistical tools
- Prepare the input (Download data, pre-process, EDA, etc.)
-   Data used: Smoking
- Classification modeling using statistical tools
-   Logistic Regression
-   Linear Discriminant Analysis
-   Quadratic Discriminant Analysis
-   Naive Bayes

Part 2: Machine-learning tools
- Prepare the input (Download data, pre-process, EDA, etc.)
-   Data used: Online Shoppers Purchasing Intention
- Classification modeling using machine-learning tools
-   K-Nearest Neighbors
-   Neural Networks
-   SVMs
-   Decision trees
-   Random Forests
-   Gradient boosting
